# MILSTD Military symbol standards in TSV and JS

This repository contains the following standards for easy reuse in other projects.

- APP-6 B

- APP-6 D

## Usage

`npm install stanag-app6`

Then you can simply:

`import { app6b,app6d } from "stanag-app6";`
